# PhyloCompare 1.0.0

First public release of PhyloCompare. PhyloCompare is a tree-comparison program that uses the PhyloFusion algorithm (
Zhang, Cetinkaya, Huson 2026, Systematic
Biology) to infer a containing network for input trees.
It then visualizes the trees through *phylogenetic parallelograms* - a novel network-derived comparison technique.
